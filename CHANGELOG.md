# Changelog

All notable changes to the Congress Tenure Tracker will be documented in this file.

## [1.0.0] - 2026-04-07

### Added
- **Tenure Simulator**: Interactive slider to simulate various term limit proposals (2-36 years).
- **Consolidated Dashboard**: High-density, data-first UI optimized for desktop and mobile.
- **Mission Statement**: Clear header verbiage explaining the site's purpose regarding the lack of term limits.
- **Party Comparison View**: Side-by-side layout for Democrats, Republicans, and Independents.
- **Vacancy Audit**: Real-time tracking of vacant seats in the House and Senate.
- **Advanced Filtering**: Search by name/state, filter by branch, tenure status, and representation type.
- **Responsive Design**: Mobile-optimized filter grid and legislator cards.
- **Sticky Metrics**: Persistent benchmark tracking while scrolling.

### Fixed
- Improved layout performance for real-time filtering of 535+ members.
- Consolidated UI to reduce vertical scrolling and improve information density.
